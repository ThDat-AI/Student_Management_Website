 📂 **Folder Structure**
```

├── backend/ # Django backend
├── frontend/ # React + Vite frontend
├── SQL/ # SQL scripts for database
└── README.md 
---
```
⚙️ **Backend Setup (Django)**

```
cd backend
python manage.py migrate
python manage.py runserver


```
⚙️ **Frontend Setup (React + Vite)**
```
cd frontend
npm install
npm run dev
```

